## A. 类型判断规则

#### A1. 帝号/谥号/庙号→人名（009-010章归纳）

帝号/谥号/庙号指代具体某人→`〖@人名〗`，不是身份`〖#〗`或官职`〖;〗`。

| 类型 | 标注 | 示例 |
|------|------|------|
| 帝号/谥号/庙号 | `〖@〗` | 高帝、高后、孝武皇帝、幽王、厉王 |
| 通称 | `〖#〗` | 天子、皇帝、太后、陛下、上（指当朝皇帝） |
| 泛称 | `〖#〗` | 今王、寡人、人主、单于 |

**判断标准**：能替换为具体人名→`〖@〗`；泛指一类身份→`〖#〗`。

#### A2. 特指后妃→人名（012章首发现）

特指某具体后妃的称号→`〖@〗`，泛称→`〖#〗`。
- `康后`（胶东康王后）→`〖@〗`；`太后`/`皇后`（泛称）→`〖#〗`

#### A3. 先帝/先王→身份（006章首发现）

"先帝""先王"是追述已故帝王的称号→`〖#身份〗`，不是官职`〖;〗`。

#### A4. 侯号作职衔→官职（011章首发现）

文中已有本名时，侯号引用→`〖;官职〗`；全章只出现侯号→可保留`〖@〗`。

#### A5. 职官+人名拆分（012章归纳）

`〖;官名〗〖;人名〗`中第二个是人名→改`〖@〗`。

#### A6. 太牢→制度（012章首发现）

太牢是祭祀规格→`〖^〗`，不是官职`〖;〗`。

#### A7. 日月三义（001章首发现）

- 天体（日月所照）→`〖!〗`；绘于器物上→`〖!〗`
- 时间纪年→`〖%〗`
- 礼制规范→`〖^〗`（极少）

#### A8. 时长vs纪年（008章归纳，001章第三轮细化）

**消歧标注规范**：

时间类型〖%〗包含两大子类，必须通过消歧标注区分：

| 子类型 | 标注格式 | 说明 |
|--------|---------|------|
| **时长/持续时间** | `〖%XX年\|时长〗` | 必须加 `\|时长` 消歧 |
| **纪年格式（年月日）** | `〖%XX年〗` | 无需消歧标注 |

**默认规则**：没有消歧标注时，默认为**纪年格式**（年月日）

**时长类型（6类，均需标注 `|时长`）**：

1. **任期时长**：在位XX年、立XX年、践帝位XX年、摄政XX年
   - `在位〖%七十载|时长〗`、`〖@尧〗立〖%七十年|时长〗得〖@舜〗`
2. **经过时长**：XX年而老/崩、XX年得、功用不成
   - `〖%二十年|时长〗而老`、`〖@尧〗於是听岳用〖@鲧〗。〖%九岁|时长〗，功用不成`
3. **守丧时长**：三年丧毕、三年四方莫举乐
   - `〖%三年|时长〗丧毕`、`〖%三年|时长〗，四方莫举乐`
4. **制度周期**：三岁一考功、五岁一巡狩
   - `〖%三岁|时长〗一考功`、`〖%五岁|时长〗一〖^巡狩〗`
5. **发展阶段**：一年成聚、二年成邑、三年成都
   - `〖%一年|时长〗而所居成聚，〖%二年|时长〗成邑，〖%三年|时长〗〖=成都〗`
6. **已经时长**：三年矣、数月矣
   - `女谋事至而言可绩，〖%三年|时长〗矣`

**纪年格式类型（3类，无需消歧）**：

1. **编年纪年**：元年、二年、三年（单独使用表某年）
   - `〖%元年〗`、`〖%三年〗`
2. **月份日期**：正月、八月、正月上日、八月丙子
   - `〖%正月上日〗，〖@舜〗受终於〖=文祖〗`
   - `〖%岁二月〗，东〖^巡狩〗；〖%五月〗，南〖^巡狩〗`
3. **完整纪年**：XX年XX月XX日
   - `〖%八月丙子〗`

**判断关键三问**：

1. **能否替换为"第X年"且不改变语义？** → 是：时长（需`|时长`）
   - "立七十年得舜" ≈ "立了七十年后得到舜" → 时长
   - "元年正月" ≠ "第元年第正月" → 纪年
2. **有语法助词"而/矣/毕"修饰？** → 是：通常是时长
   - "二十年而老"、"三年矣"、"三年丧毕"
3. **是历法/编年体的时间坐标点？** → 是：纪年格式（无需消歧）
   - "元年"、"正月上日"是时间坐标 → 纪年

**典型案例对比**：

| 原文 | 错误标注 | 正确标注 | 类型 |
|------|---------|---------|------|
| 朕在位七十载 | `朕在位〖%七十载〗` | `朕在位〖%七十载\|时长〗` | 任期时长 |
| 二十年而老 | `〖%二十年〗而老` | `〖%二十年\|时长〗而老` | 经过时长 |
| 三年丧毕 | `〖%三年〗丧毕` | `〖%三年\|时长〗丧毕` | 守丧时长 |
| 五岁一巡狩 | `〖%五岁〗一〖^巡狩〗` | `〖%五岁\|时长〗一〖^巡狩〗` | 制度周期 |
| 一年成聚 | `〖%一年〗成聚` | `〖%一年\|时长〗成聚` | 发展阶段 |
| 三年矣 | `〖%三年〗矣` | `〖%三年\|时长〗矣` | 已经时长 |
| 元年正月 | `元年〖%正月〗` | `〖%元年〗〖%正月〗` | 编年纪年 ✓ |
| 正月上日 | `〖%正月上日〗` | `〖%正月上日〗` | 月份日期 ✓ |

**注意事项**：

- 旧规律"不标〖%〗"已废弃，时长必须标注但需加`|时长`消歧
- 001章第三轮发现19处时长遗漏消歧标注
- 此规范自第三轮反思起执行

#### A11. 刑法词汇（004/006/010章）

弃市/城旦/五刑/肉刑/坑杀/屠城/菹醢/车裂/大辟/大赦→`〖[刑法〗`。

**补充（003章第二轮）**：动词形式的刑法词汇易遗漏：
- 残酷刑罚：醢（剁成肉酱）、脯（做成肉干）、剖（剖心）、车裂、菹醢
- 刑罚执行：殛（诛杀）、诛、族诛、弃市、僇
- 赦免类：赦、大赦、赦天下
- 军事动词：伐、攻、围、距（抵抗）、叛、灭
- 其他惩罚：废（废黜）、囚、俘

检查方法：`grep -E '殛|醢|脯|剖|车裂|族诛|僇|赦|伐|囚|俘|废' chapter_md/NNN_*.tagged.md`

**多义刑法动词消歧标注**（003章第二轮新增）：
刑法动词若有多义，使用消歧标注 `〖[原文|说明〗`：
- `〖[县|悬挂〗` - 悬挂展示（区别于县地名）
- `〖[封|分封〗` - 分封诸侯（区别于封土）
- `〖[封|封土〗` - 封土立墓
- `〖[释|释放〗` - 释放囚犯（区别于解释）
- `〖[表|表彰〗` - 表彰闾里（区别于表格）
- `〖[距|抵抗〗` - 军事抵抗（区别于距离）

**本纪类章节刑法密度规律**（003章第二轮新增）：
- 本纪类章节记录兴衰更替，刑法密度高于世家、列传
- 暴政段落刑法密度是全章平均的3倍
- 重点扫描："暴政段落""战争段落""更替段落"

**补充（003章第二轮）**：

- 标识类：白旗、黄旗、旗帜
- 兵器类：斧钺、弓矢、剑戟

**补充（011章第二轮）**：
- 布料等级：七布（七升布）、十五布、粗布、细布等布料规格标注为器物`〖•〗`

- "网"在捕猎语境→器物；"天罗地网"抽象比喻→不标注
- "宝玉"作为具体物品/战利品→器物；抽象品德描写→不标注

**检查方法**：
```bash
grep -nE '衣|车|琴|笠|廪|瑞|服|鼎|弓|节|玺|冠|网|宝玉|旗|斧钺' chapter_md/NNN_*.tagged.md
```

#### A18. 数量标注完整边界（001第二轮）

**错误**：`二十有〖$二人〗` → 数字前缀漏标
**正确**：`〖$二十有二人〗` → 完整数量短语

其他示例：
- `〖$三百六十六日〗`（不是`三百六〖$十六日〗`）
- `〖$五千里〗`（不是`五〖$千里〗`）

#### A19. 交通工具全覆盖（002第二轮）

**易遗漏交通工具**：船/舟/橇/暐/四载

**说明**：
- 单个交通工具：车/船/舟/橇/暐 → `〖•器物〗`
- 集合名词：四载（指车船橇暐四种工具）→ `〖•四载〗`

**标注规则**：
- **错误**：`陆行〖•乘车〗` → "乘车"不是器物名
- **正确**：`陆行乘〖•车〗` → 动词不标，只标器物

**规律**：动宾结构中，动词不纳入器物标注范围。

**检查方法**：
```bash
grep -nE '船|舟|橇|暐|四载' chapter_md/NNN_*.tagged.md
```

**注意平行段落**：如果某段描述"陆行乘车，水行乘船..."，往往会在其他段落重复出现，需对照检查。

#### A22. 典籍vs器物辨析（002第二轮）

文献名称应标注为`〖{〗`而非`〖•〗`。

**常见错误**：
- ❌ `〖•五子作歌〗`
- ✅ `〖{五子之歌〗`

#### A19. "得意"判断规则（006第二轮）

"得意"需根据语境判断：

**人名语境**：`〖@张得意〗`（明确指某个叫"得意"的人）

**动词短语语境**：不标注（表示满意、称心、实现意图）
- "明得意"：彰显志得意满
- "章得意"：显示功绩得偿所愿
- "不得意於海内"：不能称心如意於天下

**类似易误标词汇**：大业、大毕、昭明、天下大旱等也易误标为人名，需仔细辨别语境。

#### A20. "今上"→身份标注（006第二轮）

"今上"指当今皇帝，是尊称→`〖#今上〗`，不是人名。

**类似词汇**（皆为身份`〖#〗`）：
- 今上、陛下、圣上（指当今皇帝）
- 先帝、先王（指已故帝王）

**判断标准**：指代当时在位或已故帝王的通称→身份类。

#### A21. 诗歌铭文实体标注（006第二轮）

诗歌格式中的实体词汇也需完整标注，不因格式特殊而降低标注标准。

**器物类示例**：
- 戎兵、甲兵、章旗、备器等武器/旗帜
- 金石、珍宝、器械等物品

**思想类示例**：
- 仁义、礼法、王道等思想概念

**注意**：碑刻铭文是重要历史材料，标注价值高，需完整审查。

#### A23. 战争场景器物密集检查（004第二轮）

战争场景（战前誓师、交战过程、战后处置）器物密集，需逐句审查。

**关注动词**：执、持、把、握、击、射、驰、下、佩、悬、挥、载、乘、骑、驱等

**高频器物**：
- 兵器：剑、车、弓、矢、旗、钺、戈、矛、盾、甲
- 战具：戎车、兵车、战车、罕旗、常车
- 防具：甲、盾、胄

**检查方法**：
```bash
grep -nE '(执|持|把|握|击|射|驰|下|佩|悬|挥).*[剑车弓矢旗钺戈矛盾甲]' chapter_md/NNN_*.tagged.md
```

**典型段落特征**：
- 牧野之战、巨鹿之战等大型战役描写
- 单段可能出现5-10处器物遗漏
- 动词+器物的组合密集出现

#### A24. 旗名判断规则（004第二轮）

旗帜名称常见格式："颜色+旗"/"形状+旗"/"用途+旗"

**器物标注**：
- 大白旗、小白旗、罕旗、常旗 → `〖•器物〗`
- 黄旗、赤旗、黑旗、青旗 → `〖•器物〗`

**避免误标为天文**：
- 大白本是旗名，不是太白金星
- 区分方法：带"旗"字明确指旗帜→器物；单独"太白"且谈天象→天文

**检查方法**：
```bash
grep -nE '(大白|小白|罕旗|常旗|黄旗|赤旗)' chapter_md/NNN_*.tagged.md
```

#### A25. 器物省称识别（004第二轮）

在简洁语境（赞文、诗句、列举）中，器物常用省称。

**常见省称**：
- 乐 → 乐器（"太师抱乐"）
- 车 → 戎车/兵车/常车（"载以车"）
- 兵 → 兵器/甲兵
- 旗 → 某某旗（"县之旗"）

**判断方法**：
1. 查看同章其他段落是否有完整表达
2. 根据语境推断（祭祀→乐器，战争→兵器）
3. 简洁文体（赞、颂、铭）中省称频率更高

**检查方法**：在赞文段落特别关注单字器物词

#### A26. 礼器类词汇表（004第二轮）

礼仪场景（祭祀、封建、朝会）常见礼器，容易与制度混淆。

**礼器分类**：

| 类别 | 器物 | 说明 |
|------|------|------|
| 青铜器 | 鼎、彝、爵、觚、尊、卣、簋 | 祭祀用青铜礼器 |
| 玉器 | 玉、璧、圭、璋、琮、琥 | 礼仪用玉器 |
| 祭品容器 | 木主、神主、牌位 | 祭祀用牌位 |
| 祭祀用品 | 明水、牲、粟、酒、血 | 祭祀供品 |
| 仪仗 | 旗、车、剑、节 | 礼仪场合的器物 |

**易混淆点**：

#### A55. 年表"侯X"结构词性判断（021章第二轮）

`侯X不...` 结构中X的词性判断：人名 vs 副词 vs 动词。

**判断流程**：
1. 查找前文是否有"X元年" → 有：人名 / 无：继续
2. 检查X是否为常见副词 → 是：副词，不标注
3. 检查X是否为动词 → 是：不标注
4. 无法确定 → 保守不标注

**常见副词清单**：
| 副词 | 意义 | 示例 |
|------|------|------|
| 阴 | 暗中、私下 | 侯阴不使人为秋请 |
| 阳 | 公开地 | 侯阳拒之 |
| 明 | 明显地 | 侯明示其意 |
| 暗 | 暗中 | 侯暗通款曲 |

**021章案例**：
- L126: `侯〖@阴〗不使人为秋请` → `侯阴不使人为秋请`
- 分析：
  1. 前文无"阴元年"记录
  2. "阴"为常见副词（暗中）
  3. 语法："侯+副词+否定+动词"符合汉代语法
- 结论："阴"是副词，不标注

**误判原因**：
- 将"阴"理解为继承人名字
- 忽略时间逻辑：刘摇元年后仅一年即国除，无时间给继承人

**类似易混淆词**：
- 阳（副词/人名/地名）
- 明（副词/人名/形容词）

#### A59. 年表侯号官职用法判断（022章新增）

年表职官列中，侯号作为官员身份使用时标注为〖;官职〗。

**判断标准**：

| 句式 | 判断 | 标注 | 示例 |
|------|------|------|------|
| 侯号+人名+为+官职 | 侯号作官衔 | 〖;侯号〗 | 〖;绛侯〗〖@周勃〗为〖;太尉〗 |
| 封+人名+侯号 | 侯号作封号 | 〖@侯号〗 | 封〖@平津侯〗 |
| 侯号+人名（主语） | 整体是人名 | 〖@侯号人名〗 | 〖@魏其侯窦婴〗为丞相 |

**关键区分**：
- 年表职官列 → 侯号是身份标识 → 〖;〗
- 封赏/初封 → 侯号是封号 → 〖@〗
- 主语整体 → 侯号+人名不拆分 → 〖@〗

**检查方法**：
```bash

```

**错误案例**：
- ❌ `〖=曲逆〗侯〖@陈平〗为左丞相` → 误作地名
- ✅ `〖;曲逆侯〗〖@陈平〗为〖;左丞相〗` → 侯号是官衔

**适用章节**：014/018/019/020/021/022等年表类章节

#### A60. 复合动词语序保持规则（022章新增）

标注时需保证去除标注后文本语序与原文一致，避免标注破坏语序。

**典型案例**：

**"弃市"动词**：
- 原文："弃某人市"（把某人扔到市场处决）
- ❌ 错误：`〖[弃市〗〖@某人〗` → strip后"弃市某人"（语序错）
- ✅ 正确：`〖[弃〗〖@某人〗〖[市〗` → strip后"弃某人市"（语序对）

**判断方法**：
1. 标注后想象去除标记，检查字序是否与原文一致
2. 复合动词（弃市、诛族、收孥等）视语境决定拆分或整体

**022章实例**：
- 原文："十月，族灌夫家，弃魏其侯市"
- 错误标注导致：`〖[弃市〗〖@魏其侯〗` → "弃市魏其侯"（语序变）
- 正确标注：`〖[弃〗〖@魏其侯〗〖[市〗` → "弃魏其侯市"（语序对）

**验证工具**：
```bash
python scripts/lint_text_integrity.py NNN
```
应报告：0处实质差异

**易错点**：
- 复合事件连续记录（"分梁为五国，王诸子"）
- 动词+宾语紧密结构（"立为皇后"）
- 地名+动作（"汾阴得宝鼎"）

**通用原则**：标注是语义层标记，不应改变文本表面形式。

**适用章节**：全部章节，年表章节尤其易错

#### A61. 嵌套刑法标注错误（2026-03-23新发现）

**问题描述**：刑法制度名词被错误地拆分成多个独立标注，导致嵌套。

**错误模式**：

| 错误标注 | 正确标注 | 说明 |
|---------|---------|------|
| `〖[当〖[斩〗〗` | `〖[当斩〗` | "当"="应当"+"斩"被错误分别标注 |
| `〖[当〖[死〗〗` | `〖[当死〗` | "当死"="应当处死"是完整法律用语 |
| `〖[腰〖[斩〗〗` | `〖[腰斩〗` | "腰斩"是特定刑罚名称，不可拆分 |
| `〖[〖[斩〗首〗` | `〖[斩首〗` | "斩首"是完整刑罚动词 |
| `〖[系狱〗〖[当〖[死〗〗` | `〖[系狱〗〖[当死〗` | 连续嵌套错误 |

**错误根源**：
1. 标注者将"当斩"、"当死"理解为两个独立动词："当"（应当）+"斩/死"（刑罚）
2. 实际上"当斩"、"当死"是完整的法律判决用语，应作为整体标注
3. "当"字本身不是刑法动词，只是助动词

**语义分析**：
- "当斩" = "应当被斩首"（法律判决语）
- "当死" = "应当处死"（法律判决语）
- "腰斩" = 特定刑罚名称（从腰部斩断）
- "斩首" = 特定刑罚动词（斩断头颅）

**检测命令**：
```bash
grep -n '〖\[.*〖\[' chapter_md/NNN_*.tagged.md
```

**自动修复工具**：
```bash
python kg/entities/scripts/fix_nested_legal.py chapter_md/NNN_*.tagged.md
```

**修复统计**（2026-03-23）：
- 006_秦始皇本纪：3处（斩首）
- 020_建元以来侯者年表：10处（当斩4、当死5、腰斩1）
- **总计**：13处

**非错误案例**（正确的多字刑罚名词）：
- `〖[五刑〗` ✓（不是嵌套）
- `〖[流宥五刑〗` ✓（完整制度名）
- `〖[典刑〗` ✓
- `〖[肉刑〗` ✓
- `〖[砲格之刑〗` ✓

**预防建议**：
1. 遇到"当X"结构，判断"当"是否为助动词
2. 刑罚名词（斩首、腰斩、弃市等）应作为整体标注
3. 标注后检查是否存在嵌套结构

**发现渠道**：020章用户反馈 `〖[当〖[斩〗〗` 格式错误

**适用章节**：全部章节，001-020已修复

#### A62. 氏族名+谥号+王 结构标注错误（2026-03-23新发现）

**问题描述**："氏族名+谥号+王"结构被错误拆分成氏族和官职，应标注为完整人名。

**错误模式**：

| 错误标注 | 正确标注 | 说明 |
|---------|---------|------|
| `〖&长沙〗定〖;王〗` | `〖&长沙〗〖@定王\|长沙定王〗` | "定王"是完整人名（谥号+王爵） |
| `〖&河闲〗献〖;王〗` | `〖&河闲〗〖@献王\|河闲献王〗` | "河闲献王"特指某人 |
| `〖&中山〗靖〖;王〗` | `〖&中山〗〖@靖王\|中山靖王〗` | "中山靖王"是刘胜 |
| `〖&城阳〗共〖;王〗` | `〖&城阳〗〖@共王\|城阳共王〗` | "共"是谥号，非氏族部分 |

**错误根源**：
1. 将"谥号"误判为氏族名的一部分
2. 将"王"误判为官职
3. 实际上"谥号+王"是完整的人名（谥号）

**语义分析**：
- "长沙定王" = 长沙国的"定王"，谥号"定"，特指某位长沙王
- "中山靖王" = 刘胜，中山国靖王

**标注规则**：
```

```

**示例**：
- 河闲献王子 → `〖&河闲〗〖@献王|河闲献王〗〖#子〗`
- 长沙定王子 → `〖&长沙〗〖@定王|长沙定王〗〖#子〗`
- 菑川懿王子 → `〖&菑川〗〖@懿王|菑川懿王〗〖#子〗`
