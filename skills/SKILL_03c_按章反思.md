## Step 1：格式与边界扫描

### 1a. 旧格式残留和错误嵌套
- `〘X〙` → `〖+X〗`（旧生物格式）
- `@X@`、`=X=` → 对应新格式
- `〖=、〗`、`〖=。〗` → 去掉
- `<span style="...">` → 对应新格式
- ~~`⟦◈X⟧` → `〖[X〗`~~（已于2026-03-23迁移完成，001-020章无此格式）
- ~~`⟦◉X⟧` → `〖[X〗`~~（已于2026-03-23迁移完成，001-020章无此格式）
- `⟦○X⟧` → 检查是否误标，通常应去除

**⚠️ 嵌套标注错误（新发现，2026-03-23）**：
- `〖[当〖[斩〗〗` → `〖[当斩〗`（"当"和"斩"被错误地分别标注）
- `〖[当〖[死〗〗` → `〖[当死〗`（"当"和"死"被错误地分别标注）
- `〖[腰〖[斩〗〗` → `〖[腰斩〗`（"腰"和"斩"被错误地分别标注）
- `〖[〖[斩〗首〗` → `〖[斩首〗`（"斩"和"首"被错误地分别标注）
- `〖[系狱〗〖[当〖[死〗〗` → `〖[系狱〗〖[当死〗`（连续嵌套）

**错误原因**：标注者将"当斩"、"当死"等完整法律用语拆分成两个独立刑法动词，导致嵌套。
**正确理解**："当斩"="应当斩首"，是完整的法律判决用语，不应拆分。

**旧格式残留检查命令**（尤其001-012早期章节）：
```bash
grep -nE '⟦|⟧' chapter_md/NNN_*.tagged.md
```

**嵌套标注检查命令**（2026-03-23新增）：
```bash
grep -n '〖\[.*〖\[' chapter_md/NNN_*.tagged.md
```

**自动修复工具**（如果发现嵌套标注）：
```bash
python /tmp/fix_nested_legal.py chapter_md/NNN_*.tagged.md
```

### 1b. 标注边界损坏（AI生成高频错误）
| 错误模式 | 修复 |
|---------|------|
| `〖@X弟〗Y` | → `〖@X〗弟〖@Y〗` |
| `〖@，次曰〗X` | → `，次曰〖@X〗` |
| `〖@，是为〗X` | → `，是为〖@X〗` |
| `〖@弑〗`/`〖@杀〗`/`〖@攻〗` | → 去标注（动词） |
| `〖@X。〗Y` | → `〖@X〗。〖@Y〗` |
| `〖@X，〗Y` | → `〖@X〗，Y` |
| `〖@曰："〗` | → `曰："` |
| 标注吞噬后续文字 `〖•X〗，...后续〗` | 找到错位`〗`并移正 |
| 数字前缀漏标 `三百六〖$十六日〗` | `〖$三百六十六日〗` |
| 同一词重复标注 | 删除重复 |

## Step 2：现有标注审查（逐段阅读，每段都要看）

⚠️ **逐段阅读全文**，不要跳过任何段落。对每个标注判断类型和边界。

### 2a. 人名 `〖@〗` 误标（最高频错误类型）

| 误标 | 正确 | 说明 |
|------|------|------|
| 单字邦国名（赵/韩/魏/秦/齐/楚/晋/吴/越/燕/宋/卫/陈/郑/鲁等） | `〖'邦国〗` | "伐赵"/"与韩攻"→邦国；"太史赵"→人名省称 |
| `〖@X〗氏`/`〖@X〗宗`/`〖@X〗孤` | `〖&氏族〗` | 赵氏/韩氏/刘氏→氏族名 |
| 帝号/谥号/庙号 | `〖@人名〗` | 高帝/孝武/幽王→人名，不是`〖;官职〗` |
| 通称（天子/皇帝/太后/陛下） | `〖#身份〗` | 泛指不特指某人 |
| 特指后妃（康后/窦太后等） | `〖@人名〗` | 能替换为具体人→人名 |
| 侯号作职衔引用 | `〖;官职〗` | 文中已有本名时，侯号→官职 |
| 动词误入人名标注 | 去标注 | 起/杀/攻/弑/说/封/立/会/及/为 |
| `〖@帝颛顼高阳〗` | 拆分 | `〖@帝颛顼〗〖&高阳〗` |
| 长子/少子/兄/弟 | 不标注 | 亲属关系描述 |
| "以X为Y"中Y | `〖;官职〗` | 任命语境中的职位名 |

### 2b. 官职 `〖;〗` 误标

| 误标 | 正确 |
|------|------|
| 天子/皇帝/太后/皇后/陛下/先帝/先王 | `〖#身份〗` |
| 诸侯（泛称）/寡人/今王/人主/单于 | `〖#身份〗` |
| 太牢（祭祀规格） | `〖^制度〗` |
| 文学（"文学之士"的学问类别） | `〖^制度〗` |
| 太史公/褚先生（撰述者） | `〖@人名〗` |
| `〖;官名〗〖;人名〗`中第二个是人名 | 第二个改`〖@〗` |

### 2c. 时间 `〖%〗` 误标

| 误标 | 正确 |
|------|------|
| 时长（岁馀/月馀/数月/百岁/数年/二日/万岁） | 不标注 |
| 距离/人数（X里/X人/X众） | `〖$数量〗` |
| 年龄（年二十/年九十馀） | 不标注或`〖$〗` |
| 居官时长（居丞相十年） | 不标注或`〖$〗` |
| ✅ 纪年格式（元年/三年/八月丙子） | 保留`〖%〗` |

### 2d. 制度 `〖^〗` 误标

| 误标 | 正确 |
|------|------|
| 刑名（五刑/弃市/城旦/坑杀/车裂/菹醢） | `〖[刑法〗` |
| 思想概念（仁义/王道/五德/礼义/道德/天命） | `〖_思想〗` |
| 五行之德（土德/水德/火德） | `〖_思想〗` |
| 日月（天体语境） | `〖!天文〗` |

### 2e. 地名 `〖=〗` 误标

| 误标 | 正确 |
|------|------|
| 邦国名（秦/汉/齐/楚等政治实体） | `〖'邦国〗` |
| 方位副词（东北/西南/东方/西方/南方） | 不标注 |
| 泛指地貌（山川/海/野） | 不标注 |
| 九鼎等器物 | `〖•器物〗` |
| 同师（同一师父） | 不标注 |

### 2f. 语境判断（同一词汇不同含义）

| 词汇 | 语境A（标注） | 语境B（不标或改类型） |
|------|------------|-----------------|
| 日月 | `〖!〗`天体（日月所照） | `〖%〗`时间 / `〖^〗`礼制 |
| 成都 | `〖=〗`地名（四川成都） | 不标（成都邑） |
| 封禅 | `〖^〗`典章制度 | `〖:〗`具体礼仪行为 |
| 共工 | `〖@〗`人名（独立提及） | `〖;〗`官职（任命语境） |
| 符节 | `〖•〗`器物（持节/矫节） | — |
| 说 | 不标（动词，往说X） | 人名一部分（夏说/傅说） |

## Step 3：遗漏检查（按类型逐项扫描）

⚠️ 遗漏往往是修正数量最大的来源，务必仔细。

### 3a. 人名遗漏
| 句型/场景 | 说明 |
|---------|------|
| 段落内已标注人名的后续未标出现 | 同段后文的省称/再次出现 |
| 修饰语位（X大祖也/X之子） | "X"漏标 |
| 嵌入主语位（於是X妻之/乃X使之） | "X"漏标 |
| "以X为Y"中X | X 通常是人名 |

### 3b. 官职遗漏
| 句型 | 说明 |
|------|------|
| `以X为Y`（任命） | Y 是官职 |
| `拜X为Y` / `封X为Y` | Y 是官职或爵号 |
| 具体官名独立出现 | 丞相/太守/将军/御史/太尉等 |

### 3c. 地名/邦国遗漏
| 句型 | 说明 |
|------|------|
| `至于X` / `居于X` / `迁于X` | X 是地名 |
| `伐X` / `攻X` / `入X` | X 是邦国/地名 |
| 具体地名独立出现 | 咸阳/长安/邯郸等 |

### 3d. 其他类型遗漏
| 类型 | 高频遗漏 |
|------|---------|
| 生物`〖+〗` | 蝗/马/牛/鱼/鸟/犬/虎/象/龙（非神话语境）/草木 |
| 刑法`〖[〗` | 弃市/诛/坑/车裂/族诛/大辟/腰斩/大赦/赦天下 |
| 器物`〖•〗` | 金/玉/印/剑/车/旗/鼎/弓/节/玺/冠/衣 |
| 典籍`〖{〗` | "《X》曰"/"作《X》"/"传《X》" |
| 天文`〖!〗` | 星宿/历法名词/日食/彗星 |
| 氏族`〖&〗` | "姓X"/"氏X"中的X |

## 完成后操作

> ⚠️ **核心约束1：只改标注符号，不改原文字符。**
> 反思修正只允许增删 `〖〗` 标注符号，**原文汉字一个都不能增加、删除或修改**。
>
> ⚠️⚠️⚠️ **核心约束2：严禁修改动词标注格式。**
> 反思任务**绝对禁止**修改动词标注格式，动词和实体使用不同的标注符号：
> - **动词格式**：`⟦TYPE动词⟧`（数学双方括号 U+27E6/U+27E7）
> - **实体格式**：`〖TYPE内容〗`（龟甲括号 U+3016/3017）
> - **严格禁止转换**：`⟦◈⟧`/`⟦◉⟧` ↔ `〖[〗`（两种格式不可互换）
> - **格式迁移**：只能使用专门工具 `python kg/entities/scripts/migrate_verb_tags.py`
> - **参考规律**：详见 `SKILL-03c-rules.md` **规律A70**（最高优先级）
>
> **禁止操作示例**：
>
> **1. 原文字符修改（禁止）**：
> - ❌ 删字：`皆贤妇人` 中若 `〖;妇人〗` 标注有误，**不能**删掉"妇人"二字
> - ✅ 正确：去掉标注符号还原为 `皆贤妇人`
> - ❌ 增字：`族` → `〖[族诛〗`（增加了"诛"字）
> - ✅ 正确：`族` → `〖[族〗` 或 `〖[族|族诛〗`（消歧说明加到|后）
> - ❌ 增字：`斩` → `〖[斩首〗`（增加了"首"字）
> - ✅ 正确：`斩` → `〖[斩〗` 或原文就是"斩首"时才能标`〖[斩首〗`
>
> **2. 动词格式修改（严禁）**：
> - ❌❌❌ 格式转换：`⟦◈伐⟧` → `〖[伐〗`（动词格式→实体格式）
> - ❌❌❌ 格式转换：`⟦◉杀⟧` → `〖[杀〗`（动词格式→实体格式）
> - ❌❌❌ 格式转换：`〖[攻〗` → `⟦◈攻⟧`（实体格式→动词格式，只能用专用工具）
> - ✅ 正确：看到 `⟦◈⟧`/`⟦◉⟧`/`⟦○⟧` 格式 → **保持不变**
> - ✅ 正确：看到 `〖[〗` 格式 → **保持不变**（不在反思中迁移）
>
> **消歧标注规则**：
> - 如需说明词义，使用扩展标注：`〖[原文|说明〗`
> - 标注内容必须与原文字符完全一致，说明信息加到 `|` 后面

1. **应用修正**：将 A/B/C/D 中所有修正应用到 `chapter_md/NNN_章名.tagged.md`
2. **完整性验证（必须执行）**：
   ```bash
   python scripts/lint_text_integrity.py chapter_md/NNN_*.tagged.md
   ```
   结果须为 `✓ 0处实质差异`。若有差异，立即排查修复。
3. **写单章反思报告**：
   - **文件路径**：`doc/entities/第X轮按章实体反思/NNN_章名_第X轮反思报告.md`
   - **命名规范**：每章单独生成一个报告文件，即使批量反思也完成一章就生成一个
   - **报告内容**：
     - 元数据（章节、轮次、日期、修正数量）
     - 执行摘要（核心发现和主要问题类型）
     - 修正统计（按类型分类）
     - 规律应用和新发现
   - **与汇总报告的关系**：单章报告是详细记录，汇总报告引用单章报告链接
   - **历史沿革**：
     - 第一轮、第二轮：写入汇总报告 `第二轮按章实体反思报告.md`
     - 第三轮起：每章独立报告文件
4. **写回 SKILL 规律库**：E 中的新规律写入 `skills/references/SKILL-03c-rules.md` 对应类别末尾
