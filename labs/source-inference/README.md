## 🔬 研究流程

### 1. 史源溯源（SKILL_07d）
- 传承链分析：太史公怎么知道的？
- 时空可达性：信息传递路径是否合理？
- 交叉验证：其他史料如何记载？
- 史源类型判断：档案/口述/传闻

### 2. 真实性推理（SKILL_07e）
- 科学鉴定：符合医学/法律/军事常识吗？
- 逻辑推理：动机分析、排除法、反常检测
- 沉默证据：应有而未有的证据
- 临场体验：符合人性常理吗？

### 3. 可信度评级
- A级：高度可信（明确档案+多源验证）
- B级：较为可信（家传口述+逻辑合理）
- C级：存疑（时间远或细节可疑）
- D-E级：低可信/谣言（违背常识）
