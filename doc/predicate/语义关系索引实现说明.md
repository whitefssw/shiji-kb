# 语义关系索引实现说明

## 概述

在 [docs/entities/index.html](../../docs/entities/index.html) 页面添加了语义关系索引入口，与命名实体索引并列展示。语义关系索引记录《史记》全文中标注的动词性关系（⟦TYPE⟧ 格式）。

## 实现内容

### 1. 数据统计脚本

**文件**: `kg/entities/scripts/build_verb_index.py`

功能：
- 从所有 `.tagged.md` 文件中提取动词标注（⟦TYPE⟧ 格式）
- 支持消歧语法：`⟦◈伐|征伐⟧`
- 统计按章节分布、出现次数、消歧信息
- 生成 JSON 索引数据

输出：`kg/entities/data/verb_index.json`

统计结果：
- 总动作类型: **96 个**
- 总出现次数: **7151 次**
- 军事动作: 61个（4661次）
- 刑罚动作: 33个（2427次）
- 政治动作: 2个（63次）

高频动词 Top 5:
1. 杀（刑罚）: 909次
2. 伐（军事）: 775次
3. 攻（军事）: 674次
4. 击（军事）: 647次
5. 破（军事）: 455次

### 2. HTML页面生成脚本

**文件**: `kg/entities/scripts/generate_verb_html.py`

功能：
- 读取 `verb_index.json`
- 生成语义关系索引页面（4个HTML文件）

生成文件：
- `docs/entities/relations.html` - 语义关系总索引
- `docs/entities/relations-military.html` - 军事动作详细索引（95K）
- `docs/entities/relations-penalty.html` - 刑罚动作详细索引（71K）
- `docs/entities/relations-political.html` - 政治动作详细索引（3.6K）

### 3. 主索引页面更新

**文件**: `docs/entities/index.html`

变更：
- 标题从「命名实体索引」改为「知识索引」
- 增加「命名实体」和「语义关系」两个章节
- 添加4个语义关系类型卡片（总览、军事、刑罚、政治）

### 4. CSS样式扩展

**文件**: `docs/css/entity-index.css`

新增样式：
- 语义关系类型标签颜色（军事/刑罚/政治/经济）
- 语义关系卡片特殊样式
- 动词条目样式（entity-name, entity-count）
- 章节引用列表样式（chapter-ref）
- 消歧信息样式（disambiguation）
- 统计摘要框样式（stats-summary）
- 章节标题区分样式（h2, section-desc）

颜色方案：
- 语义关系总览: 皇家蓝 (#4169E1)
- 军事动作: 深红色 (#DC143C)
- 刑罚动作: 深红 (#8B0000)
- 政治动作: 钢青色 (#4682B4)
- 经济动作: 金色 (#DAA520)

## 页面结构

### 主索引页面（relations.html）

```
├─ 导航栏
├─ 标题：语义关系索引
├─ 说明文字
├─ 类型卡片网格
│  ├─ 军事（61个动作，4661次）
│  ├─ 刑罚（33个动作，2427次）
│  └─ 政治（2个动作，63次）
├─ 总体统计
└─ 导航栏
```

### 类型详细页面（如 relations-military.html）

```
├─ 导航栏
├─ 标题：军事动作索引
├─ 统计信息
├─ 类型说明框（标签 + 符号 + 描述）
├─ 搜索过滤框
├─ 动词列表
│  ├─ 动词条目1
│  │  ├─ 动词名称 + 出现次数
│  │  ├─ 章节引用列表（按出现次数降序）
│  │  └─ 消歧信息（如有）
│  ├─ 动词条目2
│  └─ ...
└─ 导航栏
```

每个动词条目显示：
- 动词名称（如"伐"）
- 总出现次数
- 章节列表（格式：`014·十二诸侯年表 (181)`）
- 消歧信息（如："征伐(120), 讨伐(35)"）

## 技术特性

### 1. 数据完整性
- 自动扫描所有 `chapter_md/*.tagged.md` 文件
- 支持动词消歧语法
- 记录章节级别的出现分布

### 2. 用户交互
- 搜索过滤功能（JavaScript实现）
- 悬停高亮效果
- 章节链接直接跳转到对应章节

### 3. 可扩展性
- 预留经济动作类型（◇）
- 类型符号体系清晰（◈军事 ◉刑罚 ○政治 ◇经济）
- JSON数据格式便于未来扩展

## 动词类型定义

基于 `kg/entities/data/verb_taxonomy.md` v3.1：

| 符号 | 类型 | 说明 | 数量 |
|------|------|------|------|
| ⟦◈⟧ | 军事 | 战争、军事行动相关 | 61个 |
| ⟦◉⟧ | 刑罚 | 刑罚执行、政治斗争 | 33个 |
| ⟦○⟧ | 政治 | 政治行为、外交活动 | 2个 |
| ⟦◇⟧ | 经济 | 经济活动、贸易行为 | 0个 |

## 使用方法

### 用户访问路径

1. 主页 → 知识索引 → 语义关系
2. 主页 → 知识索引 → 军事/刑罚/政治

```bash

# 2. 生成HTML页面
python3 kg/entities/scripts/generate_verb_html.py
```
