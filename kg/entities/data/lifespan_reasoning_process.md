## 推理方法体系

### 方法1：事件锚定法

**原理**：从已知确切年份的历史事件，推算相关人物的在位时间。

**关键锚点**：
1. **共和元年**（前841年）：中国历史确切纪年的开始
2. **城濮之战**（前632年）：晋文公五年大败楚成王
3. **殽之战**（前627年）：秦穆公三十三年攻晋失败
4. **泓之战**（前638年）：宋襄公十三年战败于楚
5. **鞍之战**（前589年）：晋景公十一年大败齐顷公
6. **邲之战**（前597年）：楚庄王十七年大败晋国
7. **问鼎中原**（前606年）：楚庄王八年问鼎轻重
8. **吴入郢**（前506年）：楚昭王十年伍子胥复仇
9. **长平之战**（前260年）：秦昭襄王四十七年

**示例推理**：

**齐桓公**：
```
已知信息：
- 《史记》记载在位43年
- 春秋五霸之首
- 管仲辅政

推理过程：
1. 根据历史记载，齐桓公约卒于前643年
2. 在位43年 → 即位年 = -643 - 43 + 1 = 前685年
3. 验证：前681年召集诸侯会盟于北杏，为即位后第5年 ✓

结果：
- 即位：前685年
- 卒年：前643年
- 在位：43年
- 置信度：high
```

**晋文公**：
```
已知信息：
- 城濮之战发生在前632年
- 《史记》记载城濮之战是文公五年
- 在位9年

推理过程：
1. 城濮之战前632年 = 文公五年
2. 即位年 = -632 + 5 - 1 = 前636年
3. 在位9年 → 卒年 = -636 - 9 + 1 = 前628年
4. 验证：《左传》记载与此吻合 ✓

结果：
- 即位：前636年
- 卒年：前628年
- 在位：9年
- 置信度：high
```

### 方法2：世系推算法

**原理**：根据父卒子继、兄终弟及的原则，由已知君主推算相邻君主。

**适用条件**：
- 明确的世系关系记载
- 相邻君主即位年份连续
- 无内乱或空位期

**示例推理**：

**齐国世系链**：
```
齐桓公（前685-前643）
  ↓ 子
齐孝公
  已知：桓公43年卒，孝公即位
  推理：即位年 = -643 + 1 = 前642年
  在位10年 → 卒年 = 前633年
  ↓ 弟
齐昭公
  已知：孝公卒后，昭公即位
  推理：即位年 = 前632年
  验证：城濮之战前632年，昭公元年 ✓
  在位19年 → 卒年 = 前613年
```

**晋国世系链**：
```
晋文公（前636-前628）
  ↓ 子
晋襄公
  已知：文公卒后襄公立
  推理：即位年 = 前627年
  验证：殽之战前627年，襄公元年 ✓
  在位7年 → 卒年 = 前621年
  ↓ 子
晋灵公
  推理：即位年 = 前620年
  在位14年 → 卒年 = 前607年（被赵穿所弑）
```

### 方法3：在位年数法

**原理**：利用《史记》明确记载的在位年数，结合已知的即位年或卒年，推算另一年份。

**公式**：
- 卒年 = 即位年 + 在位年数 - 1
- 即位年 = 卒年 - 在位年数 + 1

**示例推理**：

**楚庄王**：
```
已知信息：
- 在位23年（《史记》明确记载）
- 问鼎中原约在前606年，为庄王八年
- 春秋五霸之一

推理过程：
1. 问鼎中原前606年 = 庄王八年
2. 即位年 = -606 + 8 - 1 = 前613年
3. 在位23年 → 卒年 = -613 - 23 + 1 = 前591年
4. 验证：邲之战前597年 = 庄王17年 ✓

结果：
- 即位：前613年
- 卒年：前591年
- 在位：23年
```

**秦昭襄王**：
```
已知信息：
- 在位56年（《史记》明确记载）
- 长平之战前260年，为昭襄王四十七年

推理过程：
1. 长平之战前260年 = 昭襄王四十七年
2. 即位年 = -260 + 47 - 1 = 前306年
3. 在位56年 → 卒年 = -306 - 56 + 1 = 前251年
4. 验证：范睢任相时间与此相符 ✓

结果：
- 即位：前306年
- 卒年：前251年
- 在位：56年
```

### 方法4：交叉验证法

**原理**：通过不同国家君主之间的交往记载，相互验证推算结果的准确性。

**示例验证**：

**齐顷公与晋景公（鞍之战）**：
```
事件：鞍之战（前589年）

齐国记载：
- 顷公十年：鞍之战
- 推算：即位年 = -589 + 10 - 1 = 前598年

晋国记载：
- 景公十一年：鞍之战大败齐
- 推算：即位年 = -589 + 11 - 1 = 前599年

交叉验证：
- 齐顷公即位前598年 ✓
- 晋景公即位前599年 ✓
- 两国记载一致，相互印证
```

**楚庄王与晋景公（邲之战）**：
```
事件：邲之战（前597年）

楚国记载：
- 庄王十七年：邲之战大败晋
- 已知即位前613年
- 验证：-613 + 17 - 1 = 前597年 ✓

晋国记载：
- 景公三年：邲之战败于楚
- 已知即位前599年
- 验证：-599 + 3 - 1 = 前597年 ✓

结论：两国记载完全吻合
```
