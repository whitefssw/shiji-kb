## 🔍 使用建议

### 场景 1：首次使用
1. 阅读 [CONTENT_GUIDE.md](CONTENT_GUIDE.md) 了解整体架构
2. 浏览 [CATALOG_CN.md](CATALOG_CN.md) 快速了解所有知识单元

### 场景 2：快速查找特定主题
1. 在 [facts_index.md](facts_index.md) 或 [skills_index.md](skills_index.md) 中找到对应主题
2. 点击 SKU 链接查看详细内容

### 场景 3：深入学习技能方法
1. 在 [skills_index_enhanced.md](skills_index_enhanced.md) 中浏览技能详细说明
2. 查看应用场景、主要步骤和预期成果
3. 点击链接阅读完整 SKILL 文档

### 场景 4：探索知识关系
1. 查看 [relational_index.md](relational_index.md) 了解知识图谱结构
2. 在 [relationships_table.md](relationships_table.md) 中查找具体关系

### 场景 5：全局搜索
使用 `Ctrl+F` 或 `Cmd+F` 在文档中搜索关键词
