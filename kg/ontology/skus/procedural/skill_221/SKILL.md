# Coordinated Multi-Front Military Campaign

## Overview
Procedure for organizing and executing coordinated attacks with multiple generals leading separate forces against a common enemy.

## Steps

1. **Strategic Planning**
   - Identify the overall objective (e.g., attack Xiongnu)
   - Determine multiple departure points (上谷, 云中, 代郡, 雁门)
   - Assign equal forces to each column (e.g., 10,000 cavalry each)

2. **Assign Command Structure**
   - Appoint a supreme commander (大将军) if needed
   - Assign subordinate generals with specific titles:
     - 车骑将军
     - 轻车将军
     - 骑将军
     - 骁骑将军
   - All subordinate generals report to supreme commander

3. **Execute Independent Advances**
   - Each force advances along its assigned route
   - Forces operate independently but toward common strategic goal

4. **Report Results Separately**
   - Each general reports their own achievements and losses
   - Successes and failures are evaluated independently

5. **Evaluate Outcomes**
   - Count enemy killed and captured
   - Assess territorial gains
   - Determine rewards based on individual performance

## Key Principles
- Multiple routes prevent enemy from concentrating defense
- Independent columns allow for both success and failure without total loss
- Clear command hierarchy ensures coordination

## Expected Outcome
At least some forces will achieve objectives even if others fail; territorial gains and enemy casualties distributed across multiple fronts.

## Verification
- Confirm each column has a clearly assigned departure point, route, and commander with defined authority
- Validate that the command hierarchy (supreme commander over subordinate generals) is established and communicated
- Check that individual column results (enemy killed, captured, territory gained) are compiled and evaluated independently
